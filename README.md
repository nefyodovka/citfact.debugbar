PHP Debug Bar
=========

Отображает индикатор отладки в браузере с информацией из PHP.

![Screenshot](https://raw.github.com/nefyodovka/qbs.debugbar/master/docs/screenshot.png)

## Требования:

 - PHP версия >= 7.2
 - Bitrix версия >= 14
